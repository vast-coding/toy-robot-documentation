Install
  1. git clone my repo
  2. npm install / yarn
  3. to run robot:
    1. to run robot: npm run dev
    2. to run tests: npm run test:watch
  4. Robot instructions are in src/data/input.txt. You can edit this file, and watch console update in dev mode.
